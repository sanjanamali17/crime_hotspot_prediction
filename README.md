### Project Overview
Crime data is often available in raw tables, which makes it difficult for common users, researchers, and authorities to understand patterns and trends.
This project converts raw crime data into meaningful insights using data analysis, visualization, and an interactive web app built with Streamlit.

### The dashboard helps users:
Understand crime trends across Indian cities
Identify crime hotspots
Compare crime levels visually on a map
Gain year-wise and city-wise insights

## Problem Statement
Large crime datasets are hard to interpret
Static reports do not provide interactive exploration
No easy way to visually compare crime risk across cities

## Solution
This project provides an interactive Streamlit-based web application that:
Analyzes crime data using Python
Visualizes patterns using charts and maps
Highlights crime hotspots using color-coded risk levels
Makes data easy to understand for non-technical users
